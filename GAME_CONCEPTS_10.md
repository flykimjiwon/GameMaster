## 왜 이 10개인가? — 중독성의 7가지 과학적 원리

모든 바이럴 캐주얼 게임에는 아래 메커니즘이 최소 3개 이상 포함됨:

| 메커니즘 | 설명 | 검증 사례 |
|----------|------|-----------|
| **변동 보상** | 불확실한 보상이 고정 보상보다 도파민 400% 더 방출 | Cookie Clicker, 뱀서라이크 |
| **3초 온보딩** | 3초 안에 핵심 조작 이해 → 바이럴 임계값 | Flappy Bird, 2048 |
| **가시적 진행** | 30초마다 뭔가 성장/해금/클리어 | 아이들, 머지 |
| **실패가 재미** | 실패 순간이 공유할 만큼 극적/웃김 | Suika, Flappy Bird |
| **공유 순간** | 점수/결과를 SNS에 자연스럽게 공유 | Wordle 이모지 그리드 |
| **메타 진행** | 세션 간 영구 성장 (오프라인 진행 포함) | 뱀서라이크, Cookie Clicker |
| **감각 만족** | ASMR급 시각/청각 피드백 | Suika, 슬라이싱 게임 |

## 컨셉 1: 🍉 물리 머지 퍼즐 (Suika 변형)

### 핵심 아이디어
과일 대신 **한국 음식**(떡, 김밥, 치킨…)을 떨어뜨려 합치면 더 큰 음식이 되는 물리 퍼즐. 최종 목표: 한상차림 완성.

### 왜 중독되는가
- **Near-miss 물리**: 거의 합쳐질 뻔했는데 안 됨 → 즉시 재시도
- **ASMR 효과음**: 합쳐질 때 "뿅", "찰칵" 소리
- **스트리머 바이럴**: Suika는 VTuber 방송으로 0→400만 다운로드 (30일)

### 수익화
- 리워드 광고: "한 번 더 떨어뜨리기" 기회
- 스킨: 음식 테마 변경 (일식, 중식, 양식)

### 레퍼런스
- Suika Game: 1,100만 다운로드, iOS $200만 (3개월)
- 개발 난이도: ★★☆☆☆ (Matter.js 물리 + 간단한 머지 로직)

### 기술 스택
```
Phaser 3 + Matter.js + TypeScript
에셋: 음식 픽셀아트 (Pixelorama로 제작 or AI 생성)
사운드: jsfxr
```

## 컨셉 2: 📝 데일리 한글 퍼즐 (Wordle 한국판)

### 핵심 아이디어
하루 하나의 **4글자 사자성어/속담/유행어**를 6번 안에 맞추기. 자음/모음 힌트로 범위를 좁혀감.

### 왜 중독되는가
- **하루 1문제 = 희소성**: Wordle은 이 구조로 53억 플레이 (2024년)
- **스트릭 FOMO**: 연속 정답 기록이 깨질까 봐 매일 접속
- **이모지 공유**: 결과를 카톡/인스타에 스포일러 없이 공유

### 수익화
- 배너 광고 (하단)
- 프리미엄: 과거 문제 아카이브 ($2.99/월)
- 힌트 구매 (리워드 광고)

### 레퍼런스
- Wordle: 1,450만 DAU, NYT에 ~$100만 인수
- 개발 난이도: ★☆☆☆☆ (순수 프론트엔드, DB 불필요)

### 기술 스택
```
Next.js + TypeScript (SEO 최적화)
localStorage로 스트릭 저장
카카오톡 공유 SDK (기존 경험 활용)
```

## 컨셉 3: 🧛 뱀서라이크 (Vampire Survivors 웹 버전)

### 핵심 아이디어
자동 공격 캐릭터로 몰려드는 몬스터 웨이브를 생존. 레벨업마다 3개 중 1개 스킬 선택. 죽으면 골드로 영구 업그레이드.

### 왜 중독되는가
- **카지노 심리**: 개발자가 직접 "슬롯머신에서 영감받았다" 인정
- **실패해도 진행**: 죽어도 골드 획득 → 영구 업그레이드 → "헛된 런 없음"
- **파워 판타지**: 약한 캐릭 → 화면 가득 채우는 공격 → 쾌감 폭발

### 수익화
- 리워드 광고: 부활 1회 / 골드 2배
- 캐릭터 스킨 IAP
- 배틀 패스 ($4.99/시즌)

### 레퍼런스
- Vampire Survivors: 첫 달 $700만, 개발자 총 수익 ~$5,300만
- 개발 난이도: ★★★☆☆ (ECS 구조 + 스프라이트 관리)

### 기술 스택
```
Phaser 3 + bitECS + TypeScript
절차적 웨이브: simplex-noise로 패턴 생성
에셋: Kenney Roguelike Pack (CC0)
```

## 컨셉 4: 🍪 아이들 클리커 (K-테마)

### 핵심 아이디어
**떡볶이 가게 / 치킨집 / 카페**를 클릭으로 시작 → 자동화 → 프랜차이즈 확장. 오프라인에서도 돈이 벌림.

### 왜 중독되는가
- **숫자 올라감 중독**: Cookie Clicker는 12년 후에도 DAU 400만
- **오프라인 진행**: 앱을 안 켜도 돈이 쌓임 → 매일 확인하러 돌아옴
- **프레스티지 메커닉**: 리셋하면 배수 증가 → 또 처음부터 → 무한 루프

### 수익화
- 리워드 광고: 수익 2배 부스트 (30초)
- 프리미엄 자동화 시설 IAP
- 오프라인 수익 확장 (광고 시청)

### 레퍼런스
- Cookie Clicker: DAU 400만 (웹), Steam 200-500만
- 아이들 게임 시장: $25억 (2024), CAGR 10%
- 개발 난이도: ★★☆☆☆ (상태 관리 + 큰 숫자 처리)

### 기술 스택
```
Phaser 3 or Next.js (UI 중심이면 React도 가능)
localForage + lz-string (세이브)
BigNumber.js (큰 숫자 처리)
```

## 컨셉 5: 🎨 그림 릴레이 파티 게임 (Gartic Phone 식)

### 핵심 아이디어
제시어를 그리면 → 다음 사람이 그림만 보고 추측 → 추측을 또 다음 사람이 그림으로 → 최종 비교. **전화위복 말 전달** 게임의 그림 버전.

### 왜 중독되는가
- **못 그릴수록 재미**: 실력 없어도 오히려 웃김 → 진입장벽 0
- **매 라운드 유니크 콘텐츠**: 모든 게임이 자동으로 클립/짤 생성
- **소셜 웃음**: 보상이 게임 시스템이 아니라 사람들의 반응

### 수익화
- 전면 광고 (라운드 사이)
- 프리미엄 그리기 도구/색상 팩
- 커스텀 제시어 팩 (K-pop, 한국 음식, 영화)

### 레퍼런스
- Gartic Phone: 코로나 시기 대폭발, 스트리머 필수 게임
- 개발 난이도: ★★★☆☆ (Canvas 드로잉 + WebSocket 멀티플레이)

### 기술 스택
```
Next.js + Canvas API + Colyseus (실시간 멀티)
Socket.IO (가벼운 대안)
에셋: 최소 (드로잉 UI만)
```

## 컨셉 6: 📦 머지 & 데코 (Merge + 꾸미기)

### 핵심 아이디어
같은 아이템 3개를 머지하면 상위 아이템 생성. 머지한 아이템으로 **방/정원/카페를 꾸미기**. 꾸민 공간을 친구에게 공유.

### 왜 중독되는가
- **머지 만족감**: 합칠 때마다 시각적 보상 (이펙트 + 소리)
- **꾸미기 = 자기 표현**: 코지 게임 트렌드 (시장 $9.7억, CAGR 6.5%)
- **수집 욕구**: "다음 아이템 뭘까?" → 변동 보상
- **SNS 공유**: 예쁜 공간 스크린샷 → 인스타/카톡 바이럴

### 수익화
- IAP: 프리미엄 가구/데코 ($0.99-$4.99)
- 리워드 광고: 에너지 충전
- 시즌 이벤트: 크리스마스/벚꽃 한정 아이템

### 레퍼런스
- Merge Mansion: 수억 달러 매출
- 코지 게임 시장: $9.7억 (2024) → $14.7억 (2032)
- 여성 유저 45-55% (코지 장르)
- 개발 난이도: ★★★☆☆ (머지 로직 + 데코 배치 시스템)

### 기술 스택
```
Phaser 3 + TypeScript
드래그앤드롭: Phaser 내장 입력 시스템
에셋: Sprout Lands (itch.io) 또는 AI 생성
```

## 컨셉 7: 🏃 원터치 아케이드 (스택/점프/스윙)

### 핵심 아이디어
**탭 한 번**으로 조작하는 아케이드. 3가지 모드 번들:
1. **스택**: 블록 쌓기 (타이밍 탭)
2. **스윙**: 로프 스윙으로 플랫폼 이동 (Stickman Hook)
3. **점프**: 낭떠러지 피해 무한 점프 (Flappy 변형)

### 왜 중독되는가
- **3초 이해**: 터치 한 번이 전부. 어린이~노인 누구나
- **자기 탓 루프**: 실패가 게임 탓이 아닌 "내가 못해서" → "한 번만 더"
- **자연 난이도 상승**: 블록이 점점 작아짐 / 속도 증가

### 수익화
- 전면 광고 (3회 실패마다)
- 리워드 광고: 이어하기 / 스킨 해금
- 하이퍼캐주얼 모델: CPM 광고 최적화

### 레퍼런스
- Stack (Ketchapp): 하이퍼캐주얼 퍼즐 혁신
- Stickman Hook: Poki 상위 인기작
- Flappy Bird: 일 수익 $5만 (최고조)
- 개발 난이도: ★☆☆☆☆ (가장 빠른 개발, 1-2주)

### 기술 스택
```
Phaser 3 또는 KAPLAY (초경량)
또는 LittleJS (6KB — 모바일 최적)
에셋: 미니멀 도형 / Kenney 팩
```

## 컨셉 8: 🔮 AI 운세 + 미니게임 (사주 시너지)

### 핵심 아이디어
매일 AI가 생성하는 **오늘의 운세 + 운세 기반 미니게임**. "오늘 행운 지수 78% → 행운 테스트 게임 플레이 → 결과 공유". 기존 사주 프로젝트와 연계.

### 왜 중독되는가
- **데일리 루틴**: Wordle처럼 하루 1회 → 매일 습관
- **개인화**: "내 운세"라는 개인 맞춤 → 관여도 높음
- **공유 욕구**: "나 오늘 운세 대박" → 카톡 공유 → 바이럴
- **기존 사주 프로젝트 유저 크로스셀**: cute 앱 (내사주) → 게임 유입

### 수익화
- 프리미엄 상세 운세 (페이월)
- 리워드 광고: 럭키 스핀 추가 기회
- 크로스셀: 사주 상세 분석으로 유도

### 레퍼런스
- 한국 운세 앱 시장: 연간 수백억 원 규모
- Wordle 데일리 모델 + 사주 도메인 지식 결합
- 개발 난이도: ★★☆☆☆ (기존 사주 엔진 + 미니게임)

### 기술 스택
```
Next.js (SEO + 카카오 공유) + 사주 엔진 (@saju/shared 재사용)
미니게임: Phaser 3 임베드
LLM: GPT-4o-mini (운세 생성)
```

## 컨셉 9: 🧹 ASMR 정리/슬라이싱 (Satisfying 계열)

### 핵심 아이디어
비누 자르기, 슬라임 섞기, 방 정리하기, 모래 채우기 등 **만족스러운 감각 경험**을 제공하는 미니게임 모음.

### 왜 중독되는가
- **ASMR 도파민**: 시뮬+퍼즐이 2024년 모바일 다운로드의 **40%** (496억 중)
- **인지 부하 제로**: 생각 없이 즐김 → 다른 미디어와 병행 가능
- **틱톡 바이럴**: "oddly satisfying"은 틱톡 최고 인기 카테고리
- **모든 탭에 피드백**: 자르는 소리, 채우는 시각 효과 → 감각 보상

### 수익화
- 전면 광고 (스테이지 사이)
- 리워드 광고: 새 도구/색상 해금
- 프리미엄 스킨 팩 IAP

### 레퍼런스
- ASMR Slicing, Slice Masters: 수천만 다운로드
- 개발 난이도: ★★☆☆☆ (물리 시뮬 + 파티클 이펙트)

### 기술 스택
```
Phaser 3 + Matter.js (슬라이싱 물리)
파티클: proton-engine 또는 Phaser 내장
사운드: jsfxr + 커스텀 ASMR 효과음
```

## 컨셉 10: 🎲 소셜 추리/투표 파티 (Discord/텔레그램)

### 핵심 아이디어
AI가 생성한 **황당한 질문/시나리오**에 대해 투표하고, 누가 소수파인지 맞추기. 또는 AI가 한 명에게만 다른 제시어를 주고 누가 스파이인지 찾기.

### 왜 중독되는가
- **소셜 웃음이 보상**: 게임 시스템이 아닌 사람들의 반응이 재미
- **AI 생성 = 무한 콘텐츠**: 매번 새로운 질문/시나리오
- **제로 스킬**: 게임 실력 무관, 소통 능력만 필요
- **디스코드 90/10 수익**: 업계 최고 개발자 수익 분배

### 수익화
- 디스코드 IAP: 프리미엄 질문팩, 커스텀 테마
- 텔레그램: Stars 기반 프리미엄 모드
- 광고: CrazyGames/Poki (웹 버전)

### 레퍼런스
- Death by AI (Discord): 700만 유저, 70만 DAU
- Among Us: $1.05억 누적 (3명 개발)
- 개발 난이도: ★★★☆☆ (AI 통합 + 실시간 멀티)

### 기술 스택
```
Next.js + Colyseus (멀티플레이)
Discord Embedded App SDK / Telegram WebApp SDK
LLM: GPT-4o-mini (질문 생성)
```
