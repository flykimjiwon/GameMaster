## 5. 아트 스타일 가이드 요약

| 항목 | 가이드 |
|------|--------|
| **전체 톤** | 따뜻하고 밝은 학교/문구류 느낌. 위협적이지 않은 귀여운 전쟁 |
| **외곽선** | 2px 흑연색 `#2D2D2D` 외곽선 (손그림 느낌으로 약간 불규칙) |
| **그림자** | 드롭 섀도우 최소화. 오프셋 (2px, 2px), 색상 `#00000020` |
| **파티클** | 문구류 재질에 맞는 파티클 — 흑연 가루, 지우개 가루, 잉크 방울, 풀 방울, 금속 파편 |
| **폰트** | 제목: 손글씨 스타일 (둥근고딕/나눔손글씨). 본문: 깔끔한 고딕 |
| **애니메이션** | 스쿼시 앤 스트레치 기본 적용. 과장된 반동(recoil)으로 타격감 |
| **Tier 구분** | T1: 작고 단순, T2: 크고 디테일 추가, T3: 가장 크고 빛남(glow) + 특수 이펙트 |
| **적 피격** | 흰색 플래시 (1프레임) + 넉백. 보스는 화면 흔들림 추가 |
| **해상도** | 스프라이트 기준 48x48px. 보스 144x144px. 2x 해상도 에셋 별도 준비 |
