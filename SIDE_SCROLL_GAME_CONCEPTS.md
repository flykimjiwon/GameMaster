# 횡스크롤 게임 컨셉 기획서 (15개)

> 작성일: 2026-03-23
> 기반: GameMaster 프로젝트 (Phaser 3 + TypeScript + Vite)
> 타겟 플랫폼: Web → Poki, Telegram Mini App, Discord Activity

---

## 리서치 요약

### 시장 트렌드 (2024-2025)
- **로그라이트 융합**: Dead Cells(1000만+ 판매), Absolum(2025) 등 로그라이트+횡스크롤 조합이 지배적
- **오토배틀러 하이브리드**: Age of War 모델의 재해석, 머지+자동전투 조합 급부상
- **메트로바니아 포화**: Steam 2,200+ 타이틀, 차별화 필수
- **플랫포머 르네상스**: 2024년 Astro Bot, Prince of Persia: TLC, Neva 등 역대급 라인업
- **모바일 원탭**: Flappy Bird 아키타입의 단순 조작 + 높은 중독성 모델 여전히 유효

### 기술 스택 적합성
- Phaser 3 TileSprite + `setScrollFactor()` → 패럴랙스 구현 즉시 가능
- Arcade Physics → 2D 플랫포머 90% 커버
- Object Pooling → 무한 러너/슈팅 최적화
- 텍스처 아틀라스 → WebGL 배치 렌더링으로 모바일 60fps 달성

## 컨셉 1: 🗡️ 머지 디플로이 배틀

**장르**: 머지 + 횡스크롤 자동전투
**핵심 루프**: 머지 그리드에서 유닛 합성 → 횡스크롤 레인에 자동 배치 → 적 기지 파괴
**세션**: 2-3분

### 메카닉
- 화면 좌측: 5x3 머지 그리드 (기존 Merge Battle TD 시스템 재활용)
- 화면 우측: 횡스크롤 전투 레인 (유닛 자동 행군)
- 유닛 티어 1→2→3, 합성 시 시각적 변화 + 능력 업그레이드
- 적 웨이브: 반대편에서 진행, 시간 경과에 따라 강화

### 수익화
- 유닛 스킨 (코스메틱)
- 광고 시청 → 추가 머지 기회
- 시즌 패스 (주간 챌린지 + 한정 유닛)

### 차별점
- 기존 Merge Battle TD 코드베이스 활용으로 빠른 프로토타이핑
- 머지 만족감 + 전투 관전 재미의 이중 도파민

### 난이도: ★★☆ (기존 시스템 재활용)
### 예상 프로토타입: 2-3주

## 컨셉 2: 🎵 비트 러너 (Music Runner)

**장르**: 리듬 + 무한러너
**핵심 루프**: 음악 BPM에 맞춰 장애물 생성 → 원탭으로 점프/슬라이드 → 콤보 유지
**세션**: 1-3분 (곡 1개)

### 메카닉
- 배경 음악의 BPM 분석 → 장애물 밀도/스크롤 속도 동기화
- 박자에 맞춘 입력 → Perfect/Good/Miss 판정 → 콤보 배율
- 스테이지 = 곡 1개, 난이도 = Easy/Normal/Hard (장애물 밀도 변화)
- 캐릭터 달리기 애니메이션이 BPM에 동기화

### 아트 스타일
- Alto's Adventure 스타일 미니멀 벡터 아트
- 비트에 맞춰 배경색 전환 + 파티클 이펙트
- 실루엣 캐릭터 + 네온 트레일

### 수익화
- 곡 팩 구매 (3곡/팩)
- 캐릭터/트레일 스킨
- 커스텀 곡 업로드 (프리미엄)

### 차별점
- 음악과 게임플레이의 완벽 동기화 → 몰입감
- Veriscape + Alto's Adventure 조합은 웹에서 미개척

### 난이도: ★★★ (오디오 분석 필요)
### 예상 프로토타입: 3-4주

## 컨셉 3: 🏰 킹덤 디펜더 (Side-Scroll TD)

**장르**: 횡스크롤 타워디펜스 + 자원관리
**핵심 루프**: 말 타고 좌우 이동 → 자원 수집 → 타워 건설 → 밤에 적 방어
**세션**: 5-10분

### 메카닉
- Kingdom Two Crowns 영감: 좌우 이동하며 건설 포인트에 타워 배치
- 낮/밤 사이클: 낮=건설+자원, 밤=전투
- 3종 타워: 궁수탑(원거리), 벽(방어), 함정(범위)
- 적은 양쪽에서 공격 → 자원 배분 전략
- 웨이브 진행 시 맵 확장 (스크롤 범위 증가)

### 수익화
- 영웅 캐릭터 언락 (각각 고유 능력)
- 오프라인 자원 수집 (아이들 요소)
- 시즌 이벤트 (할로윈/크리스마스 테마 웨이브)

### 차별점
- TD + 횡스크롤 이동의 긴장감 (어디를 먼저 방어할지)
- Kingdom Two Crowns 모바일 웹 버전은 부재 → 틈새시장

### 난이도: ★★★ (맵 시스템 + 낮밤 사이클)
### 예상 프로토타입: 4-5주

## 컨셉 4: 💀 네크로 러너 (Roguelite Endless Runner)

**장르**: 로그라이트 + 무한러너
**핵심 루프**: 달리기 → 적 처치/장애물 회피 → 레벨업 시 3택 → 사망 → 메타 업그레이드 → 재도전
**세션**: 2-5분

### 메카닉
- 자동 달리기 + 점프/슬라이드/공격 3버튼
- 적 처치 → XP → 레벨업 → Vampire Survivors 스타일 3택 선택
  - 무기 추가/업그레이드/패시브 스킬
- 거리에 비례하여 난이도 상승 (적 밀도, 속도, HP)
- 사망 시 코인 획득 → 영구 업그레이드 (체력, 공격력, 시작 무기)
- 보스: 매 500m마다 등장

### 아트 스타일
- 다크 픽셀아트 (Dead Cells 느낌)
- 배경: 묘지→숲→성→지옥 (거리에 따라 바이옴 전환)

### 수익화
- 광고 시청 → 부활 1회
- 캐릭터 언락 (각각 고유 시작 무기)
- 일일 챌린지 (리더보드)

### 차별점
- 무한러너의 접근성 + 로그라이트의 리플레이 가치
- Survivor Defense 경험 활용 (레벨업/무기 시스템)

### 난이도: ★★☆ (기존 시스템 재활용)
### 예상 프로토타입: 2-3주

## 컨셉 5: 🌑 그림자 미궁 (Silhouette Metroidvania)

**장르**: 실루엣 메트로바니아
**핵심 루프**: 어두운 세계 탐험 → 능력 획득 → 새 구역 해금 → 보스 격파
**세션**: 10-30분 (세이브 포인트)

### 메카닉
- Limbo/Inside 스타일 실루엣 비주얼
- 4개 능력: 대시, 벽타기, 이중점프, 빛 구체 (어둠 밝히기)
- 빛/그림자 메카닉: 빛 구체로 숨겨진 길 발견, 그림자 속 적은 무적
- 인터커넥티드 맵: 4개 구역 (숲, 동굴, 폐허, 심연)
- 보스 4체 + 히든 보스 1체

### 아트 스타일
- 완전 실루엣 (캐릭터/적/지형)
- 배경만 컬러 그라데이션 (구역별 색상 테마)
- 파티클 이펙트로 분위기 연출 (안개, 빛줄기, 먼지)

### 수익화
- 프리미엄 모델 ($2.99-$4.99)
- 히든 챕터 DLC
- itch.io / Steam 배포

### 차별점
- 실루엣 아트 = 최소 에셋으로 최대 분위기
- 빛/그림자 메카닉으로 메트로바니아 차별화

### 난이도: ★★★★ (맵 디자인 + 능력 연동)
### 예상 프로토타입: 6-8주

## 컨셉 6: 📱 원탭 불릿헬 (One-Tap Bullet Hell Runner)

**장르**: 슈팅 + 무한러너
**핵심 루프**: 자동 전진 + 탭으로 상하 이동 → 탄막 회피 → 파워업 수집
**세션**: 1-2분

### 메카닉
- 탭: 위로 이동, 릴리스: 아래로 하강 (Flappy Bird 변형)
- 자동 사격: 전방에 총알 발사
- 적 패턴: 탄막 슈팅 패턴 (나선, 부채꼴, 조준탄)
- 파워업: 3방향탄, 레이저, 실드, 슬로우모션
- 30초마다 미니보스, 보스는 패턴 암기 필요

### 아트 스타일
- 네온 + 다크 배경 (Geometry Wars 느낌)
- 적/탄막은 밝은 색, 플레이어는 흰색
- 화면 전체가 총알로 채워지는 비주얼 임팩트

### 수익화
- 광고 시청 → 컨티뉴
- 우주선 스킨 컬렉션
- 일일/주간 리더보드

### 차별점
- 원탭 조작으로 모바일 최적화
- 탄막 슈팅의 쾌감 + 러너의 접근성

### 난이도: ★★☆
### 예상 프로토타입: 2주

## 컨셉 7: 🤜 카오스 브롤 (Co-op Beat 'em Up)

**장르**: 협동 벨트스크롤 액션
**핵심 루프**: 2-4인 온라인 → 스테이지 진행 → 적 격파 → 보스전 → 보상
**세션**: 5-10분

### 메카닉
- 4명 동시 플레이 (WebRTC P2P 또는 서버)
- 캐릭터 4종: 전사(근접), 궁수(원거리), 마법사(범위), 힐러(지원)
- 콤보 시스템: 기본공격 → 스킬 → 필살기 (게이지 축적)
- 합체기: 2인 이상 동시 필살기 → 화면 전체 공격
- 스테이지 5개 + 무한 서바이벌 모드

### 수익화
- 캐릭터 스킨 (시즌별)
- 무기 코스메틱
- Discord Activity 배포 → 디스코드 Nitro 연동

### 차별점
- 웹 기반 4인 Co-op 벨트스크롤은 거의 부재
- Discord Activity 타겟 → 소셜 바이럴

### 난이도: ★★★★★ (멀티플레이어 네트워킹)
### 예상 프로토타입: 8-10주

## 컨셉 8: 🧟 아이들 히어로 (Idle Side-Scroller)

**장르**: 방치형 + 횡스크롤 RPG
**핵심 루프**: 영웅 자동 진행 → 적 자동 처치 → 보상 수집 → 장비/스킬 업그레이드 → 반복
**세션**: 1분 (체크인) ~ 30분 (적극 플레이)

### 메카닉
- 영웅이 자동으로 우측으로 이동+공격
- 플레이어 개입: 스킬 수동 발동 (쿨다운), 장비 변경, 방향 전환
- 스테이지 시스템: 10개 적 → 보스 → 다음 스테이지
- 오프라인 보상: 접속하지 않아도 골드/XP 축적
- 전직 시스템: 100스테이지마다 전직 → 능력치 리셋 + 영구 보너스
- 펫 시스템: 동행 펫이 추가 공격/버프

### 아트 스타일
- 치비 캐릭터 + 밝은 판타지 배경
- 픽셀아트 or 벡터 (빠른 제작)

### 수익화
- 오프라인 보상 2배 (광고/구매)
- 가챠: 펫/장비 뽑기
- VIP 패스: 자동 스킬 + 배속

### 차별점
- 텔레그램 미니앱 최적화 (짧은 체크인 루프)
- 방치형이지만 횡스크롤 비주얼로 시각적 재미

### 난이도: ★★☆
### 예상 프로토타입: 3-4주

## 컨셉 9: 🐺 네바 스타일 동반자 플랫포머

**장르**: 내러티브 액션 플랫포머
**핵심 루프**: 동반자(늑대)와 함께 여정 → 전투+퍼즐 → 동반자 성장 → 감정적 스토리
**세션**: 15-30분 (챕터별)

### 메카닉
- 플레이어 캐릭터 + AI 동반자 (늑대/용/고양이)
- 동반자 성장: 아기 → 성체 (챕터 진행에 따라)
- 협동 퍼즐: 동반자에게 명령 (이동, 공격, 특수능력)
- 전투: 플레이어=근접, 동반자=지원 (점점 역할 역전)
- 4개 챕터: 봄(만남)→여름(성장)→가을(위기)→겨울(결말)

### 아트 스타일
- 수채화 느낌 (GRIS/Neva 영감)
- 계절별 색상 팔레트 전환
- 최소한의 UI, 환경으로 스토리텔링

### 수익화
- 프리미엄 ($3.99-$6.99)
- 에필로그 DLC
- 사운드트랙 판매

### 차별점
- 감정적 내러티브 = 바이럴 마케팅 (GRIS 모델)
- 웹 기반 내러티브 게임은 희소

### 난이도: ★★★★ (AI 동반자 + 내러티브 디자인)
### 예상 프로토타입: 6-8주

## 컨셉 10: ⚔️ PvP 오토배틀 레인 (Side-Scroll Auto-Battler PvP)

**장르**: PvP 자동전투 + 횡스크롤
**핵심 루프**: 유닛 편성 (30초) → 배치 → 자동 전투 관전 → 승패 → 랭크 변동
**세션**: 2-3분

### 메카닉
- 클래시 로얄 변형: 횡스크롤 뷰에서 유닛 소환
- 마나 시스템: 시간에 따라 마나 충전 → 유닛 소환
- 유닛 6종: 전사, 궁수, 기사, 마법사, 힐러, 거인
- 스펠 3종: 파이어볼(범위), 힐(아군), 프리즈(적)
- 랭크 시스템: 브론즈→실버→골드→다이아→챔피언
- 매칭: ELO 기반

### 수익화
- 유닛 스킨 (코스메틱만, P2W 없음)
- 시즌 패스 (전용 스킨 + 이모트)
- 토너먼트 입장권 (코인/광고)

### 차별점
- 클래시 로얄 + 횡스크롤 뷰 = 전투 가독성 향상
- 웹 기반 실시간 PvP (Supabase Realtime 활용)

### 난이도: ★★★★★ (실시간 PvP 서버)
### 예상 프로토타입: 8-10주

## 컨셉 11: 💣 파괴 러너 (Destructible Terrain Runner)

**장르**: 물리 기반 무한러너
**핵심 루프**: 달리기 → 폭탄/무기로 지형 파괴 → 길 만들기 → 보물 수집
**세션**: 1-3분

### 메카닉
- 자동 달리기 + 위/아래 조준 + 발사
