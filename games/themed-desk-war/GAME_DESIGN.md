# Desk War — 상세 기획서

> 책상 테마 서바이벌 디펜스 | 프로토타입 v0.1 | 1판짜리 MVP

---

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| **장르** | 뱀서류 서바이벌 디펜스 (Bullet Heaven) |
| **플랫폼** | 웹 (브라우저) → Poki, 텔레그램, 디스코드, itch.io |
| **타겟** | 남녀노소 캐주얼 유저 |
| **세션 시간** | 1판 3~5분 (프로토타입), 추후 10~30분 |
| **조작** | 이동만 (WASD/터치/마우스) |
| **엔진** | Phaser 3 + TypeScript |
| **백엔드** | Next.js + Supabase (리더보드, 세이브) |

---

## 2. 핵심 게임 루프

```
[게임 시작]
    ↓
[적 웨이브 스폰] ← 시간 경과에 따라 강해짐
    ↓
[플레이어 이동 (자동 공격)]
    ↓
[경험치 보석 수집]
    ↓
[레벨업 → 무기/스킬 3개 중 택 1] ← 핵심 선택
    ↓
[웨이브 반복... 점점 빨라지고 강해짐]
    ↓
[보스 웨이브 (60초마다)]
    ↓
[사망 or 타이머 종료 → 결과 화면]
    ↓
[코인 획득 → 영구 업그레이드 (메타 성장)]
```

## 4. 캐릭터 & 무기

### 4A. 프로토타입 캐릭터

| 속성 | 값 |
|------|-----|
| HP | 100 |
| 이동 속도 | 200px/s |
| 자석 범위 | 80px |
| 레벨업 필요 경험치 | 10 * 현재레벨 |

### 4B. 프로토타입 무기/스킬 (레벨업 선택지)

| 이름 | 타입 | 설명 |
|------|------|------|
| **회전 구슬** | 기본 무기 | 플레이어 주위를 도는 구슬. 레벨업 시 개수 증가 (1→2→3→5) |
| **번개** | 추가 무기 | 랜덤 적 1마리에 번개. 레벨업 시 대상 수 증가 |
| **HP 회복** | 패시브 | 즉시 HP 20 회복 |
| **이동 속도** | 패시브 | 이동 속도 +10% |
| **자석 범위** | 패시브 | 경험치 수집 범위 +30% |

## 8. 기술 구조

### 8A. 프로젝트 구조

```
games/survivor-defense/
├── package.json
├── vite.config.ts
├── tsconfig.json
├── index.html
├── public/
│   └── assets/
│       ├── sprites/        # 캐릭터, 적, 무기 스프라이트
│       ├── tiles/          # 맵 타일
│       └── audio/          # 효과음
└── src/
    ├── main.ts             # Phaser 게임 초기화
    ├── config.ts           # 게임 설정 상수
    ├── scenes/
    │   ├── BootScene.ts    # 에셋 로딩
    │   ├── GameScene.ts    # 메인 게임 씬
    │   ├── UIScene.ts      # HUD 오버레이 씬
    │   └── ResultScene.ts  # 결과 화면
    ├── entities/
    │   ├── Player.ts       # 플레이어 클래스
    │   ├── Enemy.ts        # 적 기본 클래스
    │   ├── EnemySpawner.ts # 웨이브 스폰 매니저
    │   └── Boss.ts         # 보스 클래스
    ├── weapons/
    │   ├── Weapon.ts       # 무기 기본 클래스
    │   ├── OrbWeapon.ts    # 회전 구슬
    │   └── LightningWeapon.ts # 번개
    ├── systems/
    │   ├── LevelUpSystem.ts  # 레벨업 선택 UI
    │   ├── XPSystem.ts       # 경험치 관리
    │   └── WaveSystem.ts     # 웨이브 타이머/난이도
    └── utils/
        └── helpers.ts
```

### 8B. 핵심 기술 포인트

| 항목 | 구현 방식 |
|------|----------|
| **렌더링** | Phaser 3 WebGL (자동 Canvas 폴백) |
| **충돌** | Phaser Arcade Physics (AABB) |
| **적 AI** | 단순 추적 (moveToObject) + 속도 변이 |
| **오브젝트 풀** | Phaser Group으로 적/투사체 풀링 (GC 방지) |
| **카메라** | Phaser Camera.startFollow(player) |
| **입력** | WASD + 터치(가상 조이스틱) + 마우스 |
| **성능 목표** | 60fps @ 500 적 동시 표시 |

### 8C. package.json 핵심 의존성

```json
{
  "dependencies": {
    "phaser": "^3.80.0"
  },
  "devDependencies": {
    "typescript": "^5.4.0",
    "vite": "^5.4.0"
  }
}
```

## 9. 에셋 계획 (프로토타입)

프로토타입은 **기하학 도형 + 단색**으로 시작, 추후 스프라이트 교체.

| 에셋 | 프로토타입 | 추후 |
|------|-----------|------|
| 플레이어 | 초록 원 | 캐릭터 스프라이트 |
| 슬라임 | 빨간 원 | 슬라임 애니메이션 |
| 박쥐 | 보라 삼각형 | 박쥐 스프라이트 |
| 골렘 | 회색 사각형 | 골렘 스프라이트 |
| 경험치 | 파란 다이아몬드 | 보석 스프라이트 |
| 회전구슬 | 흰 원 | 이펙트 |
| 번개 | 노란 선 | 번개 이펙트 |
| 맵 | 단색 배경 + 그리드 | 타일맵 |
