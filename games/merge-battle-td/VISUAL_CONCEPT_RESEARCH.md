## 5. 추천 TOP 5 (독창성 + 구현 가능성 + 에셋 확보)

### 🥇 1위: 세포/면역 생물학
- **왜?** TD에서 거의 전무한 컨셉, Merge 시스템과 완벽 매칭 (세포 합성=진화)
- **구현**: Phaser Graphics로 유기적 도형, 반투명+파스텔+발광 효과
- **에셋**: DiceBear seed 아바타 + 자체 Graphics (원/타원 기반 생물체)

### 🥈 2위: 종이접기/페이퍼크래프트
- **왜?** 시각적으로 매우 독특, "머지"=종이를 겹쳐 접기로 자연스러운 은유
- **구현**: 플랫 벡터 + 접힌 선(crease line) 오버레이, 난이도 낮음
- **에셋**: Paper Texture Pack + Paper UI Pack (무료)

### 🥉 3위: 회로기판/전자부품
- **왜?** "머지"=부품 조립, 오픈소스 참고작 존재, PCB 미학이 매우 독특
- **구현**: 녹색 보드 배경 + 금색 트레이스 경로, 부품=타워
- **에셋**: cpudefense (GitHub 오픈소스) 참고

### 4위: 칠판/낙서
- **왜?** 구현 가장 쉬움, SVG jitter로 분필 느낌, 독특한 인상
- **구현**: 검은 배경 + 흰색 떨리는 선, 지우개 이펙트
- **에셋**: 자체 Graphics만으로 완성 가능

### 5위: 보드게임/탁상
- **왜?** 디지털 TD 중 이 비주얼을 쓰는 게임 0개
- **구현**: 마분지 텍스처 + 카드/주사위 UI + Kenney Board Game Pack (490에셋)
- **에셋**: Kenney CC0 보드게임팩 바로 사용 가능
