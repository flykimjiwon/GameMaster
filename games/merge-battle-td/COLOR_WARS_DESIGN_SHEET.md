## 0. 세계관 요약

**세계에서 색이 사라지고 있다.**
암흑왕(Void King)이 세상의 모든 색을 삼키기 시작했다. 한때 찬란하던 세계는 회색 잿빛으로 변해가고, 마지막 남은 빛의 조각들만이 저항한다. **색깔 수호자**들은 빨강, 파랑, 노랑 — 빛의 3원색을 무기로 삼아 회색 몬스터들과 싸운다. 색을 혼합할수록 더 강한 힘이 깨어난다.

**핵심 메타포**: 머지 = 색의 혼합 = 힘의 결합

## 4. 타워 시트 (Towers — Merge Battle TD)

---

### 4A. 타워 공통 규칙

- **기본 형태**: 원기둥 형태의 페인트 통(paint bucket) 모티프
- **Tier 표현**: T1 = 작고 단순 (24x28px), T2 = 중간 + 장식 (28x34px), T3 = 크고 화려 + 빛남 (32x40px)
- **색 깊이**: 같은 색 머지 → 색이 더 진해짐 (채도/명도 변화), 다른 색 머지 → 새 색상

### 5E. 머지 시 색 혼합 연출

| 조합 | 연출 |
|------|------|
| 같은 색 (빨+빨) | 물감이 소용돌이치며 **더 진해지는** 애니메이션, 색상이 어두워지는 트랜지션 |
| 빨+파 → 보라 | 빨간 물감과 파란 물감이 중앙에서 만나 **보라색 소용돌이** 생성, 두 색이 섞이는 워터컬러 이펙트 |
| 빨+노 → 주황 | 빨간 불꽃과 노란 빛이 충돌하며 **주황색 폭발**, 따뜻한 톤의 파티클 |
| 파+노 → 초록 | 파란 물과 노란 빛이 합쳐지며 **초록 덩굴이 자라나는** 이펙트 |
| T3+T3 → 무지개 | 두 혼합색이 충돌 → 흰색 섬광 → **프리즘 분산** → 7색 무지개 원형 파동 → 무지개 타워 등장 |
