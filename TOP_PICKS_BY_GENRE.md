## 1. 디펜스/TD 장르

### 🥇 1위: ★ PvP 미러 디펜스 (Mirror Defense Duel)

> **"같은 재료로 서로의 타워를 뚫는 대칭 PvP"**

- **왜 1위**: 유저가 직접 지정한 1순위 + 웹 PvP 대칭 TD는 **전무 (완전 블루오션)**
- **핵심 루프**: 동일 재료 지급 → 60초 배치 → 동시 전투 → 먼저 뚫린 쪽 패배
- **중독 포인트**: 자원 배분 딜레마 (방어 vs 공격) + 심리전 + 리플레이 공유

| 항목 | 활용 리소스 |
|------|-----------|
| 엔진 | `phaser` 3.90 (설치됨) |
| 멀티플레이어 | `colyseus.js` 0.16 (설치됨) |
| 경로탐색 | `pathfinding` 0.4 (설치됨) |
| 상태관리 | `xstate` 5.28 (설치됨) — 배치/전투/결과 FSM |
| 물리 | `@dimforge/rapier2d-compat` (설치됨) — 결정론적 동기화 |
| 에셋 | **Kenney Tower Defense Kit** (CC0 무료) — 타워, 적, 맵 타일 |
| UI | `phaser3-rex-plugins` (설치됨) — 타이머, 버튼, 프로그레스바 |
| 오디오 | `howler` (설치됨) + jsfxr (웹) |
| 난이도 | ★★★★☆ |
| 예상 기간 | 4-5주 |
